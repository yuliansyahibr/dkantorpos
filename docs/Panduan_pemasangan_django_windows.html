Panduan Pemasangan D’kantorpos di Windows Server

Disiapkan oleh:

Yuliansyah Ibrahim

  1. Requirements

Dokumentasi ini dijalankan pada server yang sudah terinstal:

  • Windows server 2016
  • Python 3.7.9
  • Microsoft IIS
  • PostgreSQL 13.1

  1. Pemasangan di Microsoft IIS

  1. Install fitur CGI di IIS

  1. Download projek di https://github.com/yuliansyahibr/dkantorpos
  2. Pindahkan projek ke C:/inetpub/wwwroot/
  3. Buka CMD, install virtual environment dan dependency:

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

  1. Buat file dengan nama .env di C:\inetpub\wwwroot\dkantorpos dan isi:

secret_key="(django secret key)"

allowed_hosts="(ip server);127.0.0.1;localhost;"

db_name="(nama database)"

db_user="(username database)"

db_password="(password database)"

db_host="localhost"

db_port="5432"

email="(email)"

email_password="(password email)"

debug=False

  1. Aktivasi wfastcgi:

C:\inetpub\wwwroot\dkantorpos\venv\Scripts\wfastcgi-enable

  1. Buat file bernama web.config di C:\inetpub\wwwroot\dkantorpos dan isi:

<?xml version="1.0" encoding="utf-8"?>

<configuration>

  <system.webServer>

    <handlers>

      <add name="Python FastCGI" path="*" verb="*" modules="FastCgiModule"

      scriptProcessor="C:\inetpub\wwwroot\dkantorpos\venv\Scripts\python.exe|C:\inetpub\wwwroot\dkantorpos\venv\Lib\site-packages\wfastcgi.py" resourceType="Unspecified" requireAccess="Script" />

    </handlers>

  </system.webServer>

  <appSettings>

    <add key="PYTHONPATH" value="C:\inetpub\wwwroot\dkantorpos" />

    <add key="WSGI_HANDLER" value="dkantorpos.wsgi.application" />

    <add key="DJANGO_SETTINGS_MODULE" value="dkantorpos.settings" />

  </appSettings>

</configuration>

  1. Kumpulkan file static ke direktori STATIC_ROOT (folder assets):

python manage.py collectstatic

  1. Unlock handler di IIS:

%windir%\system32\inetsrv\appcmd unlock config -section:system.webServer/handlers

 

  1. Ubah projek menjadi default website di IIS:

%windir%\system32\inetsrv\appcmd set vdir "Default Web Site/" -physicalPath:"C:\inetpub\wwwroot\dkantorpos"

  1. Jalankan migrasi database:

python manage.py migrate

  1.  Buat file web.config di dalam folder assets dan media

<?xml version="1.0" encoding="UTF-8"?>

<configuration>

  <system.webServer>

    <handlers>

    <clear/>

      <add name="StaticFile" path="*" verb="*" modules="StaticFileModule" resourceType="File" requireAccess="Read" />

    </handlers>

  </system.webServer>

</configuration>

 

  1. Add virtual directory untuk assets dan media

  1.  Tambah permission write untuk user IIS di folder media

  1.  (Opsional) Jalankan python manage.py loaddata seed\website.json untuk import data dummy
